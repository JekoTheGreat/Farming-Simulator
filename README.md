# Farming-Simulator

This repo is for a game that I am currently working on for Engineering Programming. I intend to upload multiple variations of the game as I progress in it's development, until I finish an alpha version of the game.
It's a 2D Farming Simulator game utilizing Matracies to represent a field. I do plan to implement field expansion so the player can expand their field, though it may be expensive to do....

---- CURRENT GAME VER -- v0.0.1 ----

---- A list of features to expect ----

Harvesting
Cultivation/Plowing
Planting
Fertilizing -- Increases yield
Local Farmers Market to purchase Seeds
Game save data
Day/Night cycle -- Crops must grow, and shop keepers must sleep.
House to sleep (skip the night)
Field expansion
