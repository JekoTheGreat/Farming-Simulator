# Farming-Simulator

This repo is for a game that I am currently working on for Engineering Programming. I intend to upload multiple variations of the game as I progress in it's development, until I finish an alpha version of the game.<br />
It's a 2D Farming Simulator game utilizing Matracies to represent a field. I do plan to implement field expansion so the player can expand their field, though it may be expensive to do....<br />

---- CURRENT GAME VER -- v0.0.1 ----<br />

---- A list of features to expect ----<br />

Harvesting<br />
Cultivation/Plowing<br />
Planting<br />
Fertilizing -- Increases yield<br />
Local Farmers Market to purchase Seeds<br />
Game save data<br />
Day/Night cycle -- Crops must grow, and shop keepers must sleep.<br />
House to sleep (skip the night)<br />
Field expansion<br />
